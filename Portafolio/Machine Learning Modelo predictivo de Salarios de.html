Conclusión

El modelo de árbol indica que los factores que determinan el sueldo de un data developer son salary currency, experience level y job title.

Idealmente, para maximizar las ganancias de salario, se requiere ganar en dólares, tener 3 o más años de experiencia, y trabajar como Data Engineer o Data Scientist.
